Что такое ApcResTranslator?

Это программа для создания различных локализаций комплекса APACS3000. Перевод может быть осуществлен как вручную, так и автоматически с помощью одного из сервисов. В качестве референса используется Русский язык, т.к. изначально это основной язык комплекса (это можно указать/именить в настройках).

Поддерживаются следующие языки:

  • Азербайджанский
  • Английский
  • Казахский
  • Украинский
  • Русский

Сервисы:

  • Yandex
  • googletrans
  • GoogleTranslate

Сервисы, используемые в программе рассчитаны на коммерческое использование. Заранее ознакомьтесь с тарифами и получите токен или ключ в зависимости от сервиса.

Как этим пользоваться?

Установка API-ключа:

Чтобы получить API-ключ для Yandex.Translate, зарегистрируйте сервисный аккаунт в Яндекс.Облаке

Чтобы получить API-ключ для Google.Translate, зарегистрируйте сервисный аккаунт в Google.Cloud

Cценарий первого открытия:

Допустим у нас есть только перевод на 'Русский' для модуля Apacs30\Modules\ApcClientExtensions\ApcClExtMap. И мы хотим добавить новые языки.

  1. Нажимаем "Файл" -> "Открыть". И выбираем папку с ресурсами (у нас это ApcClExtMap\ResClMain там лежит xml файл и папки с различными локализациями). Выбираем языки, на которые нужен перевод, и референс (основной язык,в нашем случае 'русский', можно задать в настройках, чтобы постоянно не выбирать). Если нужен автоперевод, то выбираем сервис (если этому сервису нужен API-ключ, как например Yandex, задаем его в настройках). Нажимаем Ok. На этом этапе создается(если такого файла не было)/обновляется файл в формате JSON, который можно изменять в любом текстовом редакторе.
  2. Нужные поля редактируем или в самой программе, или изменяем непосредственно JSON файл в текстовом редакторе, как удобно.
  3. *Если редактировали в тестовом редакторе, можно воспользоваться консольной версией программы или нажать "Открыть из JSON".
  4. Нажимаем "Сохранить". Изменения отразились в исходных файлах ресурсов, теперь их нужно пересобрать.
  5. Нажимаем "Пересобрать ресурсы". Готово!
